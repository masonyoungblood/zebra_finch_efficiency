Zebra finches transform random songs to exhibit linguistic laws

This repository contains the code and data for “Zebra finches transform random songs to exhibit linguistic laws”, which is a re-analysis of data collected by James and Sakata (2017). The easiest way to access the code is via the interactive HTML version of the manuscript. Here are the locations of everything in the repository:

  • Code: docs/index.Rmd
  • Data: docs/data/james_sakata_2017.xlsx or https://doi.org/10.17632/sywzmd483f.1
